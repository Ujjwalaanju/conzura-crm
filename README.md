# conzura-crm
A full-stack CRM application built during my internship at Conzura Soft Solutions using React.js, Node.js, Express.js, MongoDB, and Tailwind CSS.
# Conzura CRM

A role-based Customer Relationship Management (CRM) system developed to streamline lead management, project tracking, payment management, and customer operations.

## Features

* Lead Management
* Customer Management
* Project Management
* Payment Tracking
* Task Management
* Reports & Analytics
* Role-Based Authentication
* Dashboard Overview
* Notifications

## Tech Stack

### Frontend

* React.js
* JavaScript
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Installation

### Clone the Repository

```bash
git clone https://github.com/Ujjwalaanju/conzura-crm.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the `backend` folder and configure the required environment variables before running the application.
